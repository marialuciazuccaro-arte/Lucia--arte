# Template Atelier Arte e Immagine

Questo è un sito statico dimostrativo, costruito per essere pubblicato facilmente su GitHub Pages, Netlify o Vercel.

## Pagine incluse
- Home
- Progetti
- Unità didattica: Impressionismo
- Materiali
- Galleria
- Area riservata dimostrativa

## Nota importante sull'area riservata
La pagina `area-riservata.html` NON è protetta davvero. Serve solo a mostrare l'impaginazione.
Per materiali riservati reali conviene usare link a Google Drive o Canva con permessi limitati agli account del dominio scolastico, oppure Google Sites con accesso ristretto.

## Pubblicazione rapida con GitHub Pages
1. Crea un account su GitHub.
2. Crea un nuovo repository, per esempio `atelier-arte`.
3. Carica tutti i file e le cartelle di questo pacchetto.
4. Vai su Settings > Pages.
5. Alla voce Branch scegli `main` e cartella `/root`.
6. Salva.
7. Dopo qualche minuto GitHub mostra l'indirizzo pubblico del sito.

## Come modificare i testi
Apri i file `.html` con un editor di testo. Per esempio:
- `index.html` modifica la home
- `progetti.html` modifica l'elenco dei progetti
- `unita-impressionismo.html` modifica la scheda dell'unità didattica

## Come cambiare i colori
Apri `assets/css/style.css` e modifica i valori dentro `:root`.
